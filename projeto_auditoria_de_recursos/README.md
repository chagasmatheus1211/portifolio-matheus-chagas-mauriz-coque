## 📖 Sobre o Projeto
Este projeto foi desenvolvido como parte da disciplina de Programação em Python do curso de Engenharia de Software. O objetivo do script é processar e calcular o orçamento de uma estrutura organizacional complexa utilizando dicionários aninhados, aplicando conceitos importantes da linguagem Python.

A solução foi construída utilizando recursão, funções com parâmetros dinâmicos e organização simples de código para simular o cálculo de orçamento de uma empresa.

## 🚀 Funcionalidades
- **Cálculo Hierárquico:** Varredura completa da estrutura corporativa.
- **Filtros Dinâmicos:** Possibilidade de ignorar departamentos específicos.
- **Conversão de Câmbio:** Conversão do valor final utilizando taxa de câmbio.
- **Estrutura Flexível:** Funciona com diferentes níveis de profundidade no dicionário.

## 🛠️ Tecnologias e Conceitos Aplicados
Este projeto foi construído utilizando Python puro (Standard Library), com foco nos seguintes conceitos:

* **Funções Recursivas (Recursion):** Utilizadas para navegar nos dicionários aninhados.
* **Funções com `*args`:** Para ignorar departamentos específicos.
* **Funções com `**kwargs`:** Para configurar moeda e taxa de câmbio.
* **Dicionários Aninhados:** Representando a estrutura da empresa.
