# 🎨 Image Matrix Processor: Manipulação de Dados e Pixels

## 📝 Descrição do Projeto
Este repositório contém uma suite de algoritmos em Python voltados para o processamento de estruturas de dados complexas, com foco em **manipulação de matrizes de pixels** e transformação de metadados musicais. O objetivo principal é demonstrar a aplicação de laços de repetição aninhados e lógica condicional para a modificação de ativos digitais e reorganização de bancos de dados.

O projeto foi estruturado para resolver problemas comuns em computação gráfica básica e engenharia de dados, como o ajuste de brilho/tonalidade em imagens representadas por matrizes RGB e a transposição de matrizes para readequação de interfaces de áudio.

*Figura 1: Representação visual do processamento de matrizes de pixels (Emoji Data).*

## ⚙️ Módulos e Funcionalidades

O sistema é composto por três frentes de processamento lógico:

* **🖼️ Filtro de Tonalidade (Emoji Processor):** Algoritmo que percorre matrizes RGB de emojis e aplica um filtro de escurecimento em pixels de cor amarela `(255, 255, 0)`, reduzindo sua intensidade pela metade.
* **🔄 Transposição de Matrizes (Audio Tones):** Utilitário que inverte as linhas e colunas de matrizes de toques musicais, permitindo novas formas de leitura e execução de sequências de áudio.
* **📊 Renderização de Playlists:** Integração com a biblioteca `Matplotlib` para converter dados matriciais brutos em visualizações gráficas de imagens (ex: "Sorria!").

## 🚀 Tecnologias Utilizadas
* **Linguagem:** Python 3.10
* **Bibliotecas:** Matplotlib (Visualização de Dados)
* **Ferramentas:** Google Colab, Jupyter Notebook

## 📊 Engenharia de Lógica e Aprendizados
O desenvolvimento destes módulos envolveu técnicas avançadas de manipulação de coleções:
* **Iteração em Níveis:** Uso de até 3 níveis de aninhamento (`for` loops) para acessar e modificar tuplas RGB dentro de grades.
* **Imutabilidade e Atualização:** Implementação de lógicas para criação de novas grades (`nova_grade`) e atualização de dicionários (`item.update`) para garantir a integridade dos dados originais.
* **Auditoria e Segurança:** Inclusão de um sistema de auditoria de vendas para monitorar transações anômalas e gerenciar limites de segurança globais.
