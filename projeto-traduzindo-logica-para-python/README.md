# 🛠️ Python Utility Scripts: Suite de Automação Lógica

## 📝 Descrição do Projeto
Este repositório contém uma coleção de scripts Python desenvolvidos para automatizar processos de tomada de decisão e análise de dados em diferentes domínios. O projeto foca na implementação de lógica de programação estruturada para resolver problemas cotidianos, como gestão financeira pessoal, monitoramento climático e administração acadêmica.

Os algoritmos utilizam estruturas de repetição e condicionais complexas para processar entradas do usuário em tempo real, garantindo a integridade dos dados e oferecendo feedbacks dinâmicos conforme as regras de negócio estabelecidas.

## ⚙️ Módulos do Sistema

O projeto é composto por quatro núcleos principais:

* **📊 Processamento de Vendas:** Gerencia o fechamento de carrinhos de compra, calculando subtotais e aplicando descontos progressivos (até 10%) com base no volume financeiro.
* **🌡️ Analisador de Clima:** Monitora temperaturas semanais, identifica médias e dispara alertas de segurança para condições climáticas extremas (fora da faixa de -5°C a 45°C).
* **🎓 Gestão de Notas:** Sistema de avaliação acadêmica que processa o desempenho de turmas, classificando alunos entre "Aprovado", "Recuperação" ou "Reprovado" com base em médias ponderadas.
* **💰 Simulador de Poupança:** Ferramenta de projeção financeira que calcula juros compostos mensais e permite aportes variáveis, rastreando o progresso até metas de investimento.

## 🚀 Tecnologias Utilizadas
* **Linguagem:** Python 3.10
* **Ambiente de Desenvolvimento:** Google Colab / Jupyter Notebook
* **Paradigma:** Programação Estruturada (Imperativa)

## 📊 Regras de Negócio e Aprendizados
O desenvolvimento deste conjunto de ferramentas permitiu a aplicação prática de conceitos fundamentais de Ciência da Computação:

* **Validação de Dados:** Implementação de travas para preços e quantidades negativas em sistemas de PDV.
* **Acumuladores e Contadores:** Uso de lógica de soma iterativa para cálculo de médias aritméticas em séries temporais.
* **Fluxo de Controle:** Gerenciamento de múltiplos estados de saída através de estruturas `if/elif/else` aninhadas.

2.  **Acesse o diretório do script desejado:**
    ```bash
    cd nome-do-repositorio
    ```
3.  **Execute o módulo via terminal:**
    ```bash
    python nome_do_arquivo.py
    ```

> **Nota:** Certifique-se de ter o Python 3.x instalado. Não são necessárias bibliotecas externas (como Pandas ou Numpy), pois os scripts utilizam exclusivamente a biblioteca padrão do Python.
